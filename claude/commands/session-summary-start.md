# セッションサマリー読み込みコマンド

## 目的

直近のセッションサマリーを読み込み、前回の作業を継続する。

## 実行手順

### 1. 最新のサマリーファイルを検索

`~/sandpit/` 配下の最新の summary ファイルを特定する。

```bash
find ~/sandpit -name "summary*.md" -type f 2>/dev/null | sort -r | head -1
```

### 2. サマリーを読み込む

特定したファイルの内容を読み込み、内容を把握する。

### 3. コンテキストの確認

読み込んだサマリーに基づいて、以下を確認・報告する:

- 前回のセッションで何をしていたか
- 現在の状態
- 次にやるべきこと

### 4. 作業の継続

ユーザーに確認の上、前回の続きから作業を開始する。

## 出力形式

```
## 前回のセッション (YYYY-MM-DD)

**プロジェクト**: [リポジトリ名] / [ブランチ名]

**行った作業**:
- ...

**現在の状態**:
- ...

**次にやること**:
- ...

---

前回の続きから始めますか？
```

## サマリーが見つからない場合

`~/sandpit/` にサマリーファイルが存在しない場合は、その旨を報告し、新規セッションとして開始する。
